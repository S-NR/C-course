1.2 History of C

C was developed by Dennis Ritchie in 1972 at Bell Labs. It became one of the most influential programming languages and is still widely used in operating systems, embedded systems, and high-performance software.

1.3 Applications of C
  • Operating Systems
  • Embedded Systems
  • Device Drivers
  • IoT
  • Robotics
  • Game Engines
1.4 Installing a Compiler

Install GCC (MinGW) and Visual Studio Code on Windows. Verify installation:

gcc --version
1.5 Your First C Program
#include <stdio.h>

int main()
{
    printf("Hello World\n");
    return 0;
}

This program prints Hello World and exits successfully.

1.7 Header Files

Header files provide declarations for library functions. Example: stdio.h provides printf().

1.8 printf()

printf() displays output on the console.

printf("Welcome\n");
printf("Learning C");
