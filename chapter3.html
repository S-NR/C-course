3.1 Introduction to Operators

An operator is a symbol that performs a specific operation on one or more operands (variables or values).

Example 
a + b

+ → Operator
a and b → Operands


Types of Operators

Operator TypeExamples

Relational==, !=, >, <, >=, <=
Logical&&, ||, !
Assignment=, +=, -=
Increment / Decrement++, --
Bitwise&, |, ^, ~, <<, >>

Example Program

#include<stdio.h>

int main()
{
    int a = 10;

    printf("Sum = %d", a+b);

3.4 Increment and Decrement Operators

Increment (++) and decrement (--) operators increase or decrease a variable by one.

Increment Operator (++ )

TypeDescription
Pre Increment (++a)Increment first, then use the value.
Post Increment (a++)Use the value first, then increment.

Pre Increment Example

int a = 5;
printf("%d", ++a);
6
The value of a becomes 6 before it is printed.

Post Increment Example

int a = 5;
printf("%d", a++);
5
The value is printed first. After execution, a = 6.

Decrement Operator (--)

TypeDescription
Pre Decrement (--a)Decrease first, then use.
Post Decrement (a--)Use first, then decrease.

Remember

Pre operators modify the variable before use. Post operators use the current value first.

3.5 Relational Operators

Relational operators compare two values and always return either 1 (True) or 0 (False).

OperatorDescription
>Greater Than
<Less Than
>=Greater Than or Equal
<=Less Than or Equal
==Equal To
!=Not Equal To

Example

int a = 5;
int b = 10;
printf("%d", a < b);
1

Another Example

printf("%d", 5 == 5);
1
printf("%d", 5 != 5);
0
Relational operators are mainly used in
  • if statements
  • switch statements
  • loops
  • conditional expressions

3.7 Assignment Operator

The assignment operator (=) assigns a value to a variable.

int a;
a = 25;
The value on the right side is stored in the variable on the left side.
3.8 Compound Assignment Operators

Compound assignment operators perform an operation and assignment in a single statement.

OperatorEquivalent To
+=a = a + b
-=a = a - b
*=a = a * b
/=a = a / b
%=a = a % b

Example

int a = 10;
a += 5;
printf("%d", a);
15

Equivalent Statement

a = a + 5;
