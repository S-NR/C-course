Variables are one of the most important concepts in programming. A variable is a named memory location used to store data that can change during program execution.


2.1 What is a Variable?

Think of a variable as a container or box that stores information. Instead of remembering the memory address, we give the storage location a meaningful name.


Example:

    int age = 25;

Here,

  • int → Data type
  • age → Variable name
  • 25 → Value stored

2.2 Why Do We Need Variables?
  • Store user input
  • Perform calculations
  • Store results
  • Reuse values multiple times
  • Make programs dynamic

Without variables:

    printf("10");
    printf("20");

With variables:

    int number = 10;
    printf("%d", number);
2.3 Rules for Naming Variables
Rule Example
Must begin with letter or underscore age, _total ✔
Cannot start with number 2age ✘
No spaces student_name ✔
No special characters price$ ✘
Case Sensitive Age ≠ age
Cannot use keywords int, float, return ✘
2.4 Data Types in C

A data type tells the compiler what type of data a variable can store.

Data Type Description Example
int Integer numbers 25
float Decimal numbers 10.5
double Large decimal values 12345.6789
char Single character 'A'
2.5 Declaring Variables

Syntax:

datatype variableName;
Example: 
int age;
float salary;
char grade;
2.6 Initializing Variables

Assigning a value while declaring is called initialization. 

int age = 21;
float salary = 15000.50;
char grade = 'A';
2.7 Multiple Variable Declaration
int a, b, c;

a = 10;
b = 20;
c = 30;
Or 
int a = 10, b = 20, c = 30;
2.8 Constants

A constant is a value that cannot be changed during program execution. 

const float PI = 3.14159;
Trying to change PI later will produce an error.
2.9 sizeof() Operator

The sizeof operator returns the size of a data type in bytes. 

#include

int main()
{
    printf("%d", sizeof(int));
    return 0;
}
Possible Output 
4
2.10 Format Specifiers
Specifier Used For
%d int
%f float
%lf double
%c char
%s String
Example 
int age = 20;
printf("%d", age);
2.11 Complete Program Example
#include

int main()
{
    int age = 22;
    float marks = 89.5;
    char grade = 'A';

    printf("Age = %d\n", age);
    printf("Marks = %.1f\n", marks);
    printf("Grade = %c\n", grade);

    return 0;
}
Output 
Age = 22
Marks = 89.5
Grade = A
2.12 Common Beginner Mistakes
  • Using variable before declaring it.
  • Using wrong format specifier.
  • Forgetting semicolon.
  • Using reserved keywords as variable names.
  • Not initializing variables.
